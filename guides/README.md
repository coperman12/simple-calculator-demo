## The Learning Path

### Stage 1: Developer Flow (25 minutes)

Learn the basics of running and testing code locally.

1. **[Getting Started](01-getting-started.md)** (5 min)
   - Install the project
   - Run the calculator
   - Do simple math

2. **[Understanding the Code](02-understanding-the-code.md)** (10 min)
   - See the calculator architecture
   - Learn class-based design
   - Understand error handling

3. **[Running Tests](03-running-tests.md)** (10 min)
   - Run automated tests
   - Read test reports
   - Understand pytest

---

### Stage 2: Pre-commit Hooks (10 minutes)

Learn about automatic quality checks before commits.

4. **[Pre-commit Hooks](04-pre-commit-hooks.md)** (10 min)
   - Install pre-commit hooks
   - See automatic checks in action
   - Understand quality gates

---

### Stage 3: CI/CD Workflows (15 minutes)

Learn about automated testing in the cloud.

5. **[CI/CD Basics](05-ci-cd-basics.md)** (15 min)
   - Understand GitHub Actions
   - Trigger automated tests
   - View results online

---

### Stage 4: Advanced Features (20 minutes)

Learn enterprise-level features.

6. **[Advanced Features](06-advanced-features.md)** (20 min)
   - Allure reporting with trends
   - GitHub Pages hosting
   - Performance optimization
   - Quality gates
